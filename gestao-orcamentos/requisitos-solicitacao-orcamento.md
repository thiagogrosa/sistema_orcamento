# Guia de Solicitacao de Orcamentos - Setor de Climatizacao

## Introducao

Este documento descreve todas as informacoes necessarias para solicitar um orcamento ao Setor de Orcamentos de Climatizacao. O preenchimento completo dessas informacoes garante agilidade na elaboracao e evita retrabalho.

**Importante:** Quanto mais informacoes forem fornecidas, mais rapido e preciso sera o orcamento.

- [ ] Informei se precisa de suporte
- [ ] Informei se a parte eletrica esta pronta
- [ ] Marquei os servicos complementares necessarios
- [ ] Informei o horario de atendimento

## MANUTENCAO

### Tipo de Manutencao

| Tipo | Descricao |
|------|-----------|
| Preventiva | Manutencao programada, limpeza, verificacao |
| Corretiva | Equipamento com defeito, reparo |

### Dados do(s) Equipamento(s)

| Campo | Obrigatorio | Descricao |
|-------|-------------|-----------|
| Quantidade de Equipamentos | Sim | Quantos equipamentos no escopo |
| Marca | Sim | Fabricante do equipamento |
| Modelo | Se souber | Modelo especifico |
| Capacidade (BTUs) | Sim | Capacidade de cada equipamento |
| Tipo | Sim | Split, Cassete, Piso-teto, VRF, etc. |
| Localizacao | Sim | Onde esta instalado cada equipamento |

### Para Manutencao Corretiva

| Campo | Obrigatorio | Descricao |
|-------|-------------|-----------|
| Problema Relatado | Sim | Descrever o defeito ou sintoma |
| Quando comecou? | Recomendado | Ha quanto tempo apresenta o problema |
| Ja foi feito algo? | Recomendado | Alguma tentativa de reparo anterior |

**Exemplos de problemas:**
- Nao esta gelando
- Esta vazando agua
- Faz barulho estranho
- Nao liga
- Mal cheiro
- Controle nao funciona

### Para Manutencao Preventiva (Contrato)

| Campo | Obrigatorio | Descricao |
|-------|-------------|-----------|
| Frequencia | Sim | Mensal, bimestral, trimestral, semestral |
| Duracao do Contrato | Sim | 6 meses, 12 meses, 24 meses |
| Escopo | Sim | O que deve ser incluido na manutencao |

### Checklist Manutencao

Antes de enviar a solicitacao, confirme:

- [ ] Informei o tipo (preventiva ou corretiva)
- [ ] Informei quantidade de equipamentos
- [ ] Informei marca e capacidade de cada um
- [ ] Se corretiva: descrevi o problema
- [ ] Se preventiva/contrato: informei frequencia desejada

## Resumo: O Que Enviar em Cada Caso

### Instalacao
1. Dados do cliente completos
2. Especificacao do equipamento (ou necessidade)
3. Distancia da tubulacao
4. Local da condensadora
5. Servicos complementares necessarios
6. Horario de execucao

### Manutencao
1. Dados do cliente completos
2. Tipo (preventiva/corretiva)
3. Lista de equipamentos com marca e capacidade
4. Problema relatado (se corretiva)
5. Frequencia desejada (se contrato)

### Projeto
1. Dados do cliente completos
2. Planta baixa
3. Atividade do local
4. Tipo de projeto
5. Normas aplicaveis

### Licitacao
1. Tudo acima conforme o tipo de servico
2. Numero do edital
3. Data de abertura
4. Documentos do edital

## Como Enviar a Solicitacao

### Via Asana (Preferencial)

1. Acesse o projeto **"Orcamentos - Climatizacao"**
2. Crie uma tarefa na secao **"Entrada"**
3. Titulo: `[TIPO] Cliente - Cidade` (ex: `[INSTALACAO] Empresa ABC - BH`)
4. Preencha a descricao com as informacoes deste guia
5. Anexe documentos relevantes (plantas, edital, fotos)
6. Defina o prazo
7. Salve

### Informacoes Incompletas

Se a solicitacao estiver incompleta:
- A tarefa ficara em **"Em Triagem"** aguardando complemento
- Voce sera notificado sobre quais informacoes faltam
- O prazo so comeca a contar apos informacoes completas
